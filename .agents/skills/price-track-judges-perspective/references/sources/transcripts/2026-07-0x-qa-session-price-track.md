## 七、技术环境约束（陈猛 / 省数据集团-周超 / 黄乐平 / 张瑞，要点+关键逐字）

- 评审/路演一律用**医保内网云桌面**（陈猛："路演也是使用咱们医保内部的机器。我们是提供设备，然后是大家为登录自己的账号"）。
- 云桌面/虚拟机均为 **8C16G**，虚拟机可装 MySQL/PG，存储约 200G，可申请扩容（周超）。
- **数据只进不出**（黄乐平："这是一个黑盒，相当于只进不出您工具或者说数据产品各类形式导到空间里面来"）。
- 云桌面**不允许调互联网大模型**；内网提供 **私有化部署千问 235B**，OpenAI 风格 API，统一 token 鉴权，文档含 URL/请求头/请求参数（陈猛、张瑞）。
- 无内网向量模型；需要 embedding 的要**离线下载模型→安全确认→拷入虚拟机用 CPU 跑**（陈猛与 Darren 的对话）。
- 40 张脱敏数据表，每表约1000条，四大类：医保结算/参保、门急诊住院诊疗、检查检验、院内医疗记录；其中**有结算表含药品耗材结算价格**（陈猛、刘敏）。
- 陈猛："我们提供了40张这个数据表里面都是脱敏之后的数，这个数据也不是说一定能完整的把你这个场景撑起来【…】如果是发现这个数据不足的时候，我们可以想办法就是带一些自己的数据。"（自带 mock 数据被明确允许，"因为最终评委是看到你的整体的场景的效果和你的一些亮点"）
- 代码在自己电脑开发，拷入云桌面后**改调内网 API**（陈猛："大家会调到一些互联网这种大模型，比如说千问或这些能力是吧，这个我们的云桌面是不允许直接调互联网这些算力的？但是我们在医保内网提供了一个医保内网的大模型的能力，就是这块需要大家调整一下代码"）。
- 保密协议：所有选手签字后才能领设备；数据无法导出。
